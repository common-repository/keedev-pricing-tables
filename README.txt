== Description ==
KeeDev Pricing Tables allows you to create gorgeous pricing tables for WordPress.
After installing and activating the plugin, you'll see a new "Pricing Tables" submenu under "KeeDev" menu, in which you'll be able to manage your pricing tables.

= Features: =

* Create unlimited pricing tables.
* Put your pricing tables wherever you prefer through shortcodes.
* Add price, title, text to your pricing tables.
* Sort rows as you prefer
* Start from templates
* Distraction free mode
* Choose between 2 gorgeous styles.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder and upload it into the directory `wp-content/plugins/` in your WordPress site.
3. Activate `KeeDev Pricing Tables` from Plugins page
